Kontrollspørsmål lab 2	

Forklar hvilken deler av din applikasjon så langt, implementerer de forskjellige lagene i MVC(Model-View-Controller)
arkitekturen.

Model: Dette er HTML filen, siden denne filen som er modellen og holder på dataen vi har.
View: Dette er css filen, siden den viser hvordan vår data skal se ut.
Controller: Dette er jscript, siden det er fila som kontrollerer hva som skal vises av informasjon i HTML filen.
